# Markdown
* `#` 一级标题
* `##` 二级标题
* `*` bullet point
* `** **` 加粗
* `== ==` 高亮

# Obsidian
* `ctrl + n` 新建笔记
* `ctrl + o` 切换笔记
* `ctrl + e` 切换md渲染前和渲染后视图
* `F2` 重命名文件
* `ctrl + p` 打开命令面板

# VS Code
## 1. Terminal
* ``ctrl + ` `` 显示和隐藏termianl
* ``ctrl + shift + ` ``  新建一个新的terminal
* `ctrl + shift + 5` 分屏现在的terminal（左右两边）
* `Ctrl + PgUp` / `Ctrl + PgDn`  在右侧列表中的多个独立terminal之间循环切换
* `alt + up` / `alt + down` 在关联的两个terminal之间切换
* 终端输入 `exit` 彻底关闭当前终端

## 2. 综合
* `ctrl + shift + p` 打开所有命令面板

## 3. md文件渲染
* `ctrl + shift + v` 整个md文档渲染成好看的格式，不分屏
* `ctrl + k + v` (先按k 再按v)，左右分屏看实时渲染
